# 🚚 Logistics SLA & Root Cause Analysis

## 📌 Project Overview
This project analyzes **180,000+ rows** of supply chain data to identify systemic delivery failures. By building an interactive Power BI dashboard, I moved beyond high-level metrics to perform a **Root Cause Analysis** on departmental performance.

## 📊 Key Business Insights
* **Total Impact:** Identified **103,400** late shipments across the global network.
* **The "Fan Shop" Problem:** Discovered that the Fan Shop department is the primary bottleneck, responsible for **38,240** late orders (**37%** of all failures).
* **Best Performer:** The **Fitness** department showed high operational efficiency with only **1,433** late orders.
* **Geospatial Discovery:** Identified specific high-volume regions where "Standard Class" shipping consistently fails to meet SLA targets.

## 📂 Data Source
The dataset used is the **DataCo Smart Supply Chain Systems** dataset. [Access it here on Kaggle](https://www.kaggle.com/datasets/shashwatwork/dataco-smart-supply-chain-systems).

## 🛠️ Technical Skills Demonstrated
* **Data Cleaning:** Used **Power Query** to handle null values and transform raw dates into "SLA Status" logic.
* **Data Visualization:** Built a dark-themed (Innovate) dashboard featuring KPI cards, slicers, and global maps.
* **Root Cause Analysis:** Used cross-filtering to isolate departmental failures.
